# CoopEscapeRoom

## Setup Information

Unity Version: 2019.3.11f1

There is a scripts template folder that can help with creating networked scripts. Locate the 2019.3.11f1 install folder and navigate to "Editor\Data\Resources\ScriptTemplates" and copy the scripts from the ScriptTemplates folder in the project. Once you restart Unity it will allow you to create scripts using these templates.
